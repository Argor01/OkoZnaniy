## Анимации

### Появление сообщения
```
0%   ┌─────┐  opacity: 0, translateY(10px)
     │     │
     └─────┘

50%  ┌─────┐  opacity: 0.5, translateY(5px)
     │ ... │
     └─────┘

100% ┌─────┐  opacity: 1, translateY(0)
     │ Текст│
     └─────┘
```

### Hover на кнопке отправки
```
Normal:  [📤]
Hover:   [📤]↑  scale(1.05) + shadow
Active:  [📤]↓  scale(0.95)
```

### Пульсация непрочитанных
```
0%    ●  opacity: 1
50%   ◐  opacity: 0.7
100%  ●  opacity: 1
```

## Адаптивные правила

### Скрытие элементов на мобильных
```
✅ Показывается:
- Кнопка "Назад"
- Иконки без текста
- Компактные аватары
- Основная информация

❌ Скрывается:
- Теги с ролями в сообщениях
- Кнопка загрузки файлов
- Подсказка "Enter - отправить"
- Список чатов (при открытом чате)
```

### Изменение компоновки
```
Мобильная:  flex-direction: column
Десктоп:    flex-direction: row

Мобильная:  width: 100%
Планшет:    width: 280px
Десктоп:    width: 350px
```
