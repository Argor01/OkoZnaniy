# Улучшения адаптивности чата поддержки администратора

## Что было сделано

### 1. Динамическая адаптивность
- Добавлен хук `useEffect` для отслеживания изменения размера окна в реальном времени
- Введены переменные `isMobile` (< 768px) и `isTablet` (768-1024px) для управления адаптивностью
- Все жестко заданные значения `window.innerWidth` заменены на реактивные переменные

### 2. Мобильная версия (< 768px)
- **Список чатов**: Скрывается при открытии чата, показывается кнопка "Назад"
- **Заголовок**: Сокращенный текст ("Чаты" вместо "Внутренняя коммуникация")
- **Кнопки**: Иконки без текста для экономии места
- **Участники**: Показывается до 6 аватаров вместо 8
- **Сообщения**: Уменьшенные отступы и размеры шрифтов
- **Поле ввода**: 1 строка вместо 2, скрыта кнопка загрузки файлов
- **Роли**: Скрыты теги с ролями в сообщениях для экономии места

### 3. Планшетная версия (768-1024px)
- Ширина списка чатов: 280px (вместо 350px)
- Оптимизированные отступы и размеры элементов
- Полная функциональность с адаптированными размерами

### 4. Десктопная версия (> 1024px)
- Полная ширина списка чатов: 350px
- Максимальные отступы и размеры для комфортного использования
- Все функции доступны

### 5. CSS-модуль (AdminChatsSection.module.css)
- Плавные переходы и анимации
- Адаптивные медиа-запросы
- Кастомный скроллбар
- Анимации для:
  - Появления сообщений (fadeIn)
  - Пульсации непрочитанных уведомлений
  - Индикатора онлайн-статуса
  - Hover-эффектов для кнопок и сообщений

### 6. Улучшения UX
- Плавные переходы между состояниями
- Hover-эффекты для интерактивных элементов
- Анимация отправки сообщений
- Визуальная обратная связь при взаимодействии
- Оптимизированная типографика для разных экранов

## Технические детали

### Breakpoints
```typescript
const isMobile = windowWidth < 768;
const isTablet = windowWidth >= 768 && windowWidth < 1024;
```

### Адаптивные размеры
- **Аватары**: 28px (mobile) / 32px (desktop)
- **Отступы**: 12px (mobile) / 16px (desktop)
- **Шрифты**: 13-14px (mobile) / 14-16px (desktop)
- **Кнопки**: 40px (mobile) / 44px (desktop)

### Оптимизация производительности
- Использование `React.useEffect` с cleanup для предотвращения утечек памяти
- Debounce не требуется, так как изменение размера окна происходит редко
- CSS-переходы вместо JavaScript-анимаций для лучшей производительности

## Как использовать

Компонент автоматически адаптируется к размеру экрана. Никаких дополнительных настроек не требуется.

```tsx
<AdminChatsSection
  chatRooms={chatRooms}
  currentUserId={userId}
  onSendMessage={handleSendMessage}
  onCreateRoom={handleCreateRoom}
  // ... другие пропсы
/>
```
