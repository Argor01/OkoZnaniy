# AddWorkToShop - Добавление работы в магазин

## Описание
Страница для добавления новой работы в авторский магазин готовых работ.

## Структура

```
AddWorkToShop/
├── index.tsx                    # Главный компонент страницы
├── types.ts                     # TypeScript типы
├── AddWorkToShop.module.css    # Стили страницы
├── components/
│   └── WorkForm/               # Форма добавления работы
│       ├── index.tsx
│       └── WorkForm.module.css
└── README.md
```

## Компоненты

### AddWorkToShop (главный)
- Использует Sidebar из `components/layout/Sidebar`
- Управляет навигацией и состоянием мобильного меню
- Обрабатывает сохранение и отмену формы

### WorkForm
- Форма с полями: название, цена, тип, предмет, язык
- Три RichTextEditor для: описания, оглавления, списка литературы
- Валидация обязательных полей

## Используемые компоненты

- **Sidebar** - `components/layout/Sidebar` (общий для всех страниц эксперта)
- **RichTextEditor** - `components/editor/RichTextEditor` (переиспользуемый редактор)

## Стиль
- Синий цвет: #3b82f6
- Скругленные кнопки: border-radius 8px
- Единообразный с ShopReadyWorks
