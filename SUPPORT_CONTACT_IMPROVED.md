# ✨ Улучшено окно контактов техподдержки

## Что изменено

### До:
- Текстовые эмодзи (📞, ✉️)
- Нельзя скопировать контакты
- Простой дизайн

### После:
- ✅ Красивые иконки из Ant Design
- ✅ Копирование при клике на телефон/email
- ✅ Hover эффекты
- ✅ Уведомления о копировании
- ✅ Современный дизайн

## Новые возможности

### 📞 Телефон
- Иконка: `PhoneOutlined` (зелёная)
- При клике: копируется номер `88005007857`
- Уведомление: "Телефон скопирован!"

### ✉️ Email
- Иконка: `MailOutlined` (синяя)
- При клике: копируется email `b-oko.znaniy@mail.ru`
- Уведомление: "Email скопирован!"

### 🕐 График работы
- Иконка: `ClockCircleOutlined` (оранжевая)
- Информация: Пн-Пт 07:00 - 16:00 (МСК)

### 📋 Иконка копирования
- Показывается справа от каждого контакта
- Подсказывает, что можно скопировать

## Технические детали

### Файл изменён:
`frontend-react/src/components/common/DashboardHeader.tsx`

### Добавленные иконки:
```typescript
import {
  PhoneOutlined,      // Телефон
  MailOutlined,       // Email
  ClockCircleOutlined, // Часы
  CopyOutlined,       // Копирование
} from '@ant-design/icons';
```

### Функция копирования:
```typescript
onClick={(e) => {
  e.stopPropagation();
  navigator.clipboard.writeText('88005007857');
  message.success('Телефон скопирован!');
}}
```

### Hover эффекты:
```typescript
onMouseEnter={(e) => {
  e.currentTarget.style.background = '#f3f4f6';
}}
onMouseLeave={(e) => {
  e.currentTarget.style.background = '#f9fafb';
}}
```

## Дизайн

### Цвета:
- Фон карточки: `#f9fafb`
- Фон при hover: `#f3f4f6`
- Телефон: `#10b981` (зелёный)
- Email: `#3b82f6` (синий)
- Часы: `#f59e0b` (оранжевый)
- Иконка копирования: `#9ca3af` (серый)

### Размеры:
- Минимальная ширина: 280px
- Padding карточек: 8px 12px
- Border radius: 8px
- Размер иконок: 16px (основные), 12px (копирование)

## Как проверить

1. Открой http://localhost:5173
2. Войди как клиент (test_client / test123)
3. Кликни на иконку наушников в хедере
4. Увидишь обновлённое окно с красивыми иконками
5. Кликни на телефон - он скопируется
6. Кликни на email - он скопируется
7. Увидишь уведомления о копировании
