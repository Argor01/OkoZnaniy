## Анимации:

### fadeIn (появление компонента):
```css
0%   → opacity: 0
100% → opacity: 1
Длительность: 0.3s
```

### fadeInUp (появление сообщений):
```css
0%   → opacity: 0, translateY(10px)
100% → opacity: 1, translateY(0)
Длительность: 0.3s
```

### Hover-эффекты:
```css
Сообщения: translateY(-2px) + усиление тени
Кнопка отправки: scale(1.1) + усиление тени
Кнопка файла: scale(1.05) + изменение цвета
Список чатов: translateX(4px)
```
