# Исправления системы уведомлений

## Выполненные изменения

### 1. Исправлен z-index модального окна уведомлений
- Модальное окно теперь отображается поверх сайдбара
- z-index увеличен до 100001 (сайдбар имеет 100000)
- Применено к `.ant-modal-wrap`, `.ant-modal-mask` и `.ant-modal`

### 2. Добавлена адаптивность для модального окна

#### Мобильные устройства (≤840px):
- Модальное окно занимает весь экран (100vw, 100vh)
- Убраны отступы и скругления
- Уменьшены размеры иконок (36px вместо 40px)
- Уменьшены шрифты (14px/13px)
- Настройки отображаются в колонку
- Адаптивная высота контента

#### Планшеты (841px - 1024px):
- Модальное окно занимает 90% ширины экрана
- Сохранены все функции десктопной версии

### 3. Убраны локальные уведомления

#### ExpertDashboardRefactored.tsx:
- Удалены локальные тестовые данные `notifications`
- Добавлен хук `useNotifications()` для загрузки данных из БД
- Уведомления теперь загружаются через API
- Добавлена функция `markAsRead()` при клике на уведомление
- Счетчик непрочитанных использует `unreadCount` из хука

#### DashboardWithChatAndNotifications.tsx:
- Уже использует хук `useNotifications()`
- Все данные загружаются из БД
- Нет локальных данных

## API для уведомлений

Используется API из `src/api/notifications.ts`:
- `GET /notifications/` - получить все уведомления
- `POST /notifications/{id}/mark_read/` - отметить как прочитанное
- `POST /notifications/mark_all_read/` - отметить все как прочитанные

## Типы уведомлений

- `order` - уведомления о заказах
- `claim` - претензии
- `message` - сообщения
- `balance` - операции с балансом
- `bid` - ставки на заказы
- `system` - системные уведомления

## Автообновление

Уведомления автоматически обновляются каждые 30 секунд через хук `useNotifications`.
