# Страница "Партнерская программа" в ЛК партнера

## Что добавлено

### Новый компонент PartnerProgram
- **Файл**: `frontend-react/src/features/partner/pages/PartnerProgram.tsx`
- **Стили**: `frontend-react/src/features/partner/pages/PartnerProgram.module.css`

### Функциональность

1. **Описание программы**
   - Заголовок и краткое описание партнерской программы
   - Информация о монетизации образовательного трафика

2. **Партнерская ссылка**
   - Отображение реферальной ссылки
   - Кнопка копирования в буфер обмена
   - Счетчик поделившихся в соцсетях

3. **Способы продвижения**
   - 5 основных методов продвижения с иконками:
     - Рекомендации в соцсетях и блогах
     - Использование баннеров и виджетов
     - Привлечение заказчиков и исполнителей
     - Увеличение базы партнеров
     - Заработок с каждого заказа

4. **Условия выплат**
   - Таблица с процентами отчислений:
     - С заказчиков: 25%
     - С исполнителей: 5%
   - Условия повторных заказов
   - Продолжительность выплат
   - Важные примечания о разделении начислений
   - Информация о промокодах

5. **Бонусная программа**
   - Описание дополнительных выплат (1000₽ за каждые 10000₽)
   - Карточки текущей и следующей выплаты
   - Прогресс-бар до следующего бонуса
   - Отслеживание в реальном времени

### Интеграция в PartnerDashboard

1. **Добавлен новый пункт меню**:
   ```typescript
   {
     key: 'program',
     icon: <InfoCircleOutlined />,
     label: 'Партнерская программа',
     component: <PartnerProgram />
   }
   ```

2. **Изменен порядок меню**:
   - Партнерская программа (по умолчанию)
   - Статистика
   - Мои рефералы
   - История начислений

3. **Передача данных**:
   - `referralLink` - ссылка партнера
   - `currentEarnings` - текущие заработки
   - `nextBonusThreshold` - порог для следующего бонуса

### Адаптивность

- **Мобильные устройства**: Адаптивная сетка, уменьшенные шрифты
- **Планшеты**: Оптимизированное расположение карточек
- **Десктоп**: Полная функциональность

### Стилизация

- **CSS модули** для изоляции стилей
- **Цветовая схема** Ant Design
- **Иконки** для каждого раздела
- **Hover эффекты** для интерактивных элементов
- **Адаптивная типографика**

## Использование

### Для разработчиков

```typescript
import { PartnerProgram } from '@/features/partner/pages/PartnerProgram';

<PartnerProgram 
  referralLink="https://studwork.org/ref/ABC123"
  currentEarnings={5000}
  nextBonusThreshold={10000}
/>
```

### Для пользователей

1. Войти в ЛК партнера
2. По умолчанию откроется страница "Партнерская программа"
3. Скопировать реферальную ссылку
4. Изучить условия выплат
5. Отслеживать прогресс бонусной программы
