# Карта партнеров

## Описание

Карта партнеров отображает географическое распределение партнеров по городам России. Данные загружаются в реальном времени из базы данных.

## Функциональность

- Отображение партнеров на интерактивной карте России
- Фильтрация партнеров по городам
- Просмотр детальной информации о каждом партнере
- Статистика по рефералам и доходам

## Использование

### Для партнеров

1. Войдите в личный кабинет партнера
2. Перейдите в раздел "Карта партнеров"
3. Используйте фильтр по городам для поиска партнеров в конкретном регионе
4. Кликните на маркер или карточку партнера для просмотра подробной информации

### Для администраторов

#### Создание тестовых партнеров

Для тестирования карты можно создать тестовых партнеров с помощью команды:

```bash
python manage.py create_test_partners
```

Эта команда создаст партнеров в следующих городах:
- Москва
- Санкт-Петербург
- Новосибирск
- Екатеринбург
- Краснодар
- Владивосток
- Казань

Все тестовые партнеры будут иметь пароль: `testpass123`

## Технические детали

### Поддерживаемые города

Карта поддерживает более 70 городов России, включая:
- Все города-миллионники
- Крупные региональные центры
- Города федерального значения

Полный список городов с координатами находится в файле `PartnersMap.tsx`.

### API Endpoint

Данные партнеров загружаются через endpoint:
```
GET /users/partners_list/
```

Требуется аутентификация с ролью `partner`.

### Структура данных

Каждый партнер содержит следующие поля:
- `id` - уникальный идентификатор
- `username` - имя пользователя
- `email` - электронная почта
- `city` - город (обязательно для отображения на карте)
- `phone` - телефон (опционально)
- `total_referrals` - общее количество рефералов
- `active_referrals` - количество активных рефералов
- `total_earnings` - общий доход в рублях
- `date_joined` - дата регистрации

### Добавление новых городов

Чтобы добавить поддержку нового города:

1. Откройте файл `frontend-react/src/features/partner/pages/PartnersMap.tsx`
2. Найдите константу `CITY_COORDINATES`
3. Добавьте новый город с его координатами (широта, долгота):

```typescript
'Название города': { x: широта, y: долгота },
```

Пример:
```typescript
'Тюмень': { x: 57.1522, y: 65.5272 },
```

## Устранение неполадок

### Партнер не отображается на карте

Проверьте:
1. Указан ли город в профиле партнера
2. Есть ли координаты для этого города в `CITY_COORDINATES`
3. Правильно ли написано название города (с учетом регистра)

### Карта не загружается

1. Проверьте наличие файла `russia-map.json` в публичной директории
2. Убедитесь, что API endpoint `/users/partners_list/` доступен
3. Проверьте права доступа пользователя (должна быть роль `partner`)

## Разработка

### Структура компонента

```
PartnersMap.tsx
├── Загрузка данных карты (russia-map.json)
├── Загрузка данных партнеров (API)
├── Преобразование координат (географические → SVG)
├── Рендеринг карты
│   ├── SVG карта России
│   └── Маркеры партнеров
└── Боковая панель со списком партнеров
```

### Стили

Стили находятся в файле `PartnersMap.module.css`.
