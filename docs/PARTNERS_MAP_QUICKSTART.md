# Быстрый старт: Карта партнеров

## Что изменилось

Карта партнеров теперь использует **реальные данные из базы данных** вместо тестовых данных.

## Как проверить

### 1. Проверьте текущих партнеров

```bash
python manage.py list_partners
```

Эта команда покажет всех партнеров в базе данных.

### 2. Удалите старых тестовых партнеров (если есть)

```bash
python manage.py cleanup_test_partners
```

Эта команда найдет и удалит старых тестовых партнеров с именами типа "Тестовый партнер 1", "test@example.com" и т.д.

### 3. Создайте новых тестовых партнеров

```bash
python manage.py create_test_partners
```

Эта команда создаст 7 партнеров в разных городах России с тестовыми данными.

### 4. Войдите как партнер

Используйте любой из созданных аккаунтов:
- Логин: `partner_moscow`, `partner_spb`, `partner_novosibirsk`, и т.д.
- Пароль: `testpass123`

### 5. Откройте карту партнеров

Перейдите в раздел "Карта партнеров" в личном кабинете партнера.

## Что работает

✅ Загрузка реальных данных партнеров из API  
✅ Отображение партнеров на карте России  
✅ Фильтрация по городам  
✅ Просмотр детальной информации о партнере  
✅ Статистика по рефералам и доходам  
✅ Поддержка 70+ городов России  

## Что удалено

❌ Тестовые данные (hardcoded mock data)  
❌ Fallback на фейковых партнеров  

## Добавление реальных партнеров

Чтобы добавить реального партнера:

1. Создайте пользователя с ролью `partner`
2. Укажите город в профиле партнера
3. Убедитесь, что город есть в списке поддерживаемых (см. `CITY_COORDINATES` в `PartnersMap.tsx`)

## Добавление нового города

Если нужно добавить город, которого нет в списке:

1. Откройте `frontend-react/src/features/partner/pages/PartnersMap.tsx`
2. Найдите `CITY_COORDINATES`
3. Добавьте строку:
   ```typescript
   'Название города': { x: широта, y: долгота },
   ```

Координаты можно найти на [Google Maps](https://www.google.com/maps) или [Яндекс.Картах](https://yandex.ru/maps).
