# Инструкция по тестированию партнерских реферальных ссылок

## Шаг 1: Создание тестового партнера

```bash
python manage.py shell
```

```python
from apps.users.models import User

# Создаем партнера
partner = User.objects.create_user(
    username='testpartner',
    email='partner@test.com',
    password='test123',
    role='partner',
    first_name='Тест',
    last_name='Партнеров'
)

print(f"✅ Партнер создан!")
print(f"Username: testpartner")
print(f"Password: test123")
print(f"Реферальный код: {partner.referral_code}")
print(f"Ссылка: http://localhost:5173/login?ref={partner.referral_code}")
```

## Шаг 2: Вход в ЛК партнера

1. Откройте http://localhost:5173/login
2. Войдите:
   - Username: `testpartner`
   - Password: `test123`
3. Перейдите в раздел "Партнерская программа" → "Промоматериалы"
4. Откройте вкладку "Партнерская ссылка"

**Что должно отображаться:**
- ✅ Реферальный код (например: `ABC12345`)
- ✅ Партнерская ссылка (например: `http://localhost:5173/login?ref=ABC12345`)
- ✅ Кнопки "Копировать" для кода и ссылки
- ✅ Статистика: Всего рефералов: 0, Активных: 0, Комиссия: 5%

## Шаг 3: Тестирование реферальной ссылки

### 3.1 Прямая ссылка

1. Скопируйте партнерскую ссылку из ЛК
2. Откройте её в режиме инкогнито или другом браузере
3. Вы должны попасть на страницу http://localhost:5173/login

**Проверьте:**
- ✅ Страница загрузилась
- ✅ Есть вкладки "Зарегистрироваться" и "Войти"
- ✅ Переключитесь на вкладку "Зарегистрироваться"
- ✅ Должно отображаться уведомление о реферальном коде (зеленая плашка)
- ✅ В форме должно быть скрытое поле с реферальным кодом

### 3.2 Короткие ссылки

Проверьте, что все эти ссылки работают и перенаправляют на `/login`:

```
http://localhost:5173/ref/ABC12345
http://localhost:5173/referral/ABC12345
http://localhost:5173/r/ABC12345
```

Каждая должна:
1. Показать экран загрузки "Переход по реферальной ссылке..."
2. Перенаправить на `/login?ref=ABC12345`
3. Сохранить код в localStorage

## Шаг 4: Регистрация нового пользователя

На странице `/login?ref=ABC12345`:

1. Переключитесь на вкладку "Зарегистрироваться"
2. Заполните форму:
   - Email: `newuser@test.com`
   - Username: `newuser`
   - Password: `test123`
   - Роль: Клиент или Исполнитель
3. Нажмите "Зарегистрироваться"

**Ожидаемый результат:**
- ✅ Регистрация прошла успешно
- ✅ Пользователь автоматически вошел в систему
- ✅ Реферальный код был применен

## Шаг 5: Проверка привязки в базе данных

```python
from apps.users.models import User

# Найдите нового пользователя
new_user = User.objects.get(username='newuser')

# Проверьте привязку
print(f"✅ Новый пользователь: {new_user.username}")
print(f"✅ Email: {new_user.email}")
print(f"✅ Партнер: {new_user.partner}")
print(f"✅ Имя партнера: {new_user.partner.first_name} {new_user.partner.last_name}")
print(f"✅ Реферальный код партнера: {new_user.partner.referral_code}")

# Проверьте статистику партнера
partner = User.objects.get(username='testpartner')
print(f"\n📊 Статистика партнера:")
print(f"✅ Всего рефералов: {partner.total_referrals}")
print(f"✅ Активных рефералов: {partner.active_referrals}")

# Список всех рефералов
print(f"\n👥 Список рефералов:")
for ref in partner.referrals.all():
    print(f"  - {ref.username} ({ref.email}) - роль: {ref.role}")
```

**Ожидаемый вывод:**
```
✅ Новый пользователь: newuser
✅ Email: newuser@test.com
✅ Партнер: testpartner
✅ Имя партнера: Тест Партнеров
✅ Реферальный код партнера: ABC12345

📊 Статистика партнера:
✅ Всего рефералов: 1
✅ Активных рефералов: 0

👥 Список рефералов:
  - newuser (newuser@test.com) - роль: client
```

## Шаг 6: Проверка обновления статистики в ЛК

1. Вернитесь в ЛК партнера (обновите страницу)
2. Перейдите в "Промоматериалы" → "Партнерская ссылка"
3. Проверьте статистику:
   - ✅ Всего рефералов: 1
   - ✅ Активных: 0 (станет 1 после первого заказа)

## Шаг 7: Тестирование копирования

1. В ЛК партнера нажмите "Копировать" рядом с реферальным кодом
2. Должно появиться уведомление "Скопировано в буфер обмена"
3. Вставьте в любое текстовое поле - должен быть код (например: `ABC12345`)

4. Нажмите "Копировать" рядом с партнерской ссылкой
5. Должно появиться уведомление "Скопировано в буфер обмена"
6. Вставьте в браузер - должна быть полная ссылка (например: `http://localhost:5173/login?ref=ABC12345`)

## Возможные проблемы и решения

### Проблема: Реферальный код не отображается у партнера

**Решение:**
```python
from apps.users.models import User
import uuid

partner = User.objects.get(username='testpartner')
if not partner.referral_code:
    partner.referral_code = str(uuid.uuid4())[:8].upper()
    partner.save()
    print(f"Код создан: {partner.referral_code}")
```

### Проблема: Реферальный код не подставляется на странице регистрации

**Проверьте:**
1. Откройте консоль браузера (F12)
2. Проверьте localStorage: `localStorage.getItem('referral_code')`
3. Проверьте URL: должен быть параметр `?ref=CODE`
4. Проверьте, что нет ошибок в консоли

### Проблема: Новый пользователь не привязывается к партнеру

**Проверьте в Django shell:**
```python
from apps.users.models import User

# Проверьте, что партнер существует
partner = User.objects.filter(referral_code='ABC12345', role='partner').first()
print(f"Партнер найден: {partner}")

# Проверьте нового пользователя
new_user = User.objects.get(username='newuser')
print(f"Партнер у нового пользователя: {new_user.partner}")
```
