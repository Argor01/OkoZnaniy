# Инструкция по настройке VK-бота для уведомлений

## Что нужно сделать

Чтобы бот заработал, нам нужно получить **два значения** из ВКонтакте:
- **Токен сообщества** (длинный код, по которому бот будет отправлять сообщения)
- **ID сообщества** (номер вашей группы ВК)

Ниже — пошаговая инструкция со скриншотами, как это получить.

---

## Шаг 1. Создайте сообщество ВКонтакте (если его ещё нет)

1. Откройте ВКонтакте: https://vk.com
2. В левом меню нажмите **«Сообщества»**
3. Нажмите кнопку **«Создать сообщество»** (в правом верхнем углу)
4. Выберите тип — **«Бизнес»** или **«Бренд или организация»**
5. Введите название, например: **OkoZnaniy — Уведомления**
6. Нажмите **«Создать сообщество»**

---

## Шаг 2. Узнайте ID сообщества

1. Откройте только что созданное сообщество
2. Нажмите **«Управление»** (значок шестерёнки справа)
3. Прокрутите вниз — в адресной строке браузера будет ссылка вида:
   ```
   https://vk.com/club123456789
   ```
4. **Число после "club"** — это и есть ID сообщества. В примере выше это `123456789`

**Другой способ:**
1. В управлении сообществом откройте раздел **«Работа с API»**
2. ID сообщества будет указан на этой странице

> 📌 **Запишите это число** — оно нам понадобится.

---

## Шаг 3. Включите сообщения сообщества

1. В управлении сообществом откройте **«Сообщения»**
2. Переключите **«Сообщения сообщества»** → **Включены**
3. Нажмите **«Сохранить»**

Без этого шага бот не сможет отправлять сообщения пользователям.

---

## Шаг 4. Включите возможности ботов (Long Poll API)

1. В управлении сообществом перейдите в **«Работа с API»**
2. Откройте вкладку **«Long Poll API»**
3. Включите **Long Poll API** (переключатель → Включено)
4. Убедитесь, что выбрана **версия API: 5.199** (или самая свежая)
5. В разделе **«Типы событий»** поставьте галочки:
   - ✅ **Входящее сообщение** (message_new)
   - ✅ Остальные события можно оставить по умолчанию
6. Нажмите **«Сохранить»**

---

## Шаг 5. Создайте ключ доступа (токен)

1. В разделе **«Работа с API»** откройте вкладку **«Ключи доступа»**
2. Нажмите **«Создать ключ»**
3. В открывшемся окне отметьте галочками:
   - ✅ **Разрешить приложению доступ к сообщениям сообщества**
   - ✅ **Разрешить приложению управление сообществом**
4. Нажмите **«Создать»**
5. Подтвердите действие (может потребоваться ввести пароль или код из SMS)
6. Появится длинный код — это **токен сообщества**

> ⚠️ **ВАЖНО:** Скопируйте токен сразу и сохраните в надёжном месте!
> После закрытия окна его нельзя будет посмотреть снова — только создать новый.

Токен выглядит примерно так:
```
vk1.a.AbCdEfGhIjKlMnOpQrStUvWxYz1234567890...
```

---

## Шаг 6. Отправьте данные разработчику

После выполнения всех шагов у вас должны быть **два значения**:

| Что | Пример | Где взяли |
|-----|--------|-----------|
| **ID сообщества** | `123456789` | Шаг 2 — число из адреса группы |
| **Токен сообщества** | `vk1.a.AbCdEf...` | Шаг 5 — ключ доступа |

Отправьте оба значения разработчику удобным способом.

> 🔒 **Токен — это как пароль.** Не публикуйте его в открытых чатах и не отправляйте в общие группы. Передавайте только лично разработчику.

---

## Готово!

После получения этих данных разработчик:
1. Впишет их в настройки сервера
2. Запустит бота
3. Бот начнёт работать — отправлять уведомления пользователям в ВК

### Как пользователи подключат бота:
- Пользователю достаточно **написать боту** в ВК сообщение **«Начать»**
- Бот автоматически привяжет их аккаунт
- После этого уведомления с сайта будут приходить прямо в ВК

---

## Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Не вижу раздел «Работа с API» | Убедитесь, что вы — администратор сообщества |
| Не могу включить сообщения | Проверьте, что у вас есть права администратора |
| Забыл скопировать токен | Создайте новый ключ доступа (Шаг 5) |
| Бот не отвечает пользователям | Проверьте, что сообщения сообщества включены (Шаг 3) |
