2. ✅ Добавлена полная адаптивность для мобильных и планшетов

## Как протестировать

### 1. Проверка z-index (модалка поверх сайдбара)

1. Откройте дашборд эксперта
2. Откройте мобильное меню (сайдбар)
3. Нажмите на иконку уведомлений
4. **Ожидаемый результат**: Модальное окно уведомлений должно отображаться поверх сайдбара

### 2. Проверка адаптивности

#### Мобильные устройства (≤840px):
1. Откройте DevTools (F12)
2. Переключитесь в режим мобильного устройства
3. Откройте уведомления
4. **Ожидаемый результат**: 
   - Модалка занимает весь экран
   - Нет отступов по краям
   - Контент читаемый и удобный
   - Настройки отображаются в колонку

#### Планшеты (841px - 1024px):
1. Установите ширину экрана 900px
2. Откройте уведомления
3. **Ожидаемый результат**: Модалка занимает 90% ширины экрана

#### Десктоп (>1024px):
1. Установите ширину экрана 1920px
2. Откройте уведомления
3. **Ожидаемый результат**: Модалка имеет фиксированную ширину 700px

### 3. Проверка загрузки из БД

1. Откройте дашборд эксперта (ExpertDashboardRefactored)
2. Откройте уведомления
3. Откройте DevTools → Network
4. **Ожидаемый результат**: 
   - Должен быть запрос к `/api/notifications/`
   - Уведомления загружаются с сервера
   - Нет локальных тестовых данных

### 4. Проверка отметки как прочитанное

1. Откройте уведомления
2. Кликните на непрочитанное уведомление (с синей точкой)
3. **Ожидаемый результат**:
   - Запрос к `/api/notifications/{id}/mark_read/`
   - Уведомление помечается как прочитанное
   - Счетчик непрочитанных уменьшается

### 5. Проверка автообновления

1. Откройте уведомления
2. Подождите 30 секунд
3. **Ожидаемый результат**: Автоматический запрос к API для обновления списка
